Displaying "Hello World" on a 0.96" OLED Display Using Arduino

Introduction

OLED (Organic Light Emitting Diode) displays are widely used in embedded systems for their low power consumption and high contrast. This tutorial will guide you through displaying "Hello World" on a 0.96-inch SSD1306 OLED display using an Arduino board.

Components Required

  • Arduino Uno/Nano (or any compatible board)
  • 0.96" I2C OLED Display (SSD1306)
  • Jumper Wires
  • Breadboard (Optional)

Note: On different microcontrollers like ESP32, the I2C pins may be different, so always verify the correct I2C pins for your board.

How OLED SSD1306 Works

The SSD1306 OLED is a monochrome display that communicates via I2C (Inter-Integrated Circuit) protocol. It requires only two data lines (SDA & SCL) to communicate with the microcontroller. The default I2C address for most 0.96-inch OLEDs is 0x3C, but some modules use 0x3D. If the display does not work, you may need to scan for the I2C address and adjust accordingly.

Installing Required Libraries

To interface the SSD1306 OLED with Arduino, we need to install the necessary libraries:

  1. Open Arduino IDE.
  2. Go to Sketch → Include Library → Manage Libraries.
  3. Search and install the following libraries:
  • Adafruit GFX Library
  • Adafruit SSD1306 Library
  1. Restart Arduino IDE after installation.

Arduino Code to Display "Hello World"

Below is the complete Arduino code to display "Hello World" on the OLED screen:

#include <Wire.h>

#include <Adafruit_GFX.h>

#include <Adafruit_SSD1306.h>

#define SCREEN_WIDTH 128 // OLED display width, in pixels

#define SCREEN_HEIGHT 64 // OLED display height, in pixels

// Declaration for an SSD1306 display connected to I2C (SDA, SCL pins)

Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, -1);

void setup() {

  Serial.begin(115200);

  display.begin(SSD1306_SWITCHCAPVCC, 0x3C);

  display.clearDisplay();

 

  display.setTextSize(2);

  display.setTextColor(WHITE);

  display.setCursor(10, 20);

  display.print("Hello World");

  display.display();

}

void loop() {

  // Nothing here as we only need to display text once

}

Explanation of the Code

  1. Library Inclusion:
  • The Wire.h, Adafruit_GFX.h, and Adafruit_SSD1306.h libraries are included to interface with the OLED display.
  1. OLED Initialization:
  • The OLED screen resolution is defined as 128x64.
  • The display is initialized with display.begin(SSD1306_SWITCHCAPVCC, 0x3C).
  1. Displaying Text:
  • The display is cleared using display.clearDisplay().
  • The text size is set to 2 for better visibility.
  • The text is positioned using display.setCursor(10, 20).
  • "Hello World" is displayed using display.print().
  1. Refreshing the Display:
  • display.display() updates the OLED screen to show the text.
  1. Loop Function:
  • The loop() function is empty because we only need to display the text once.

Uploading the Code to Arduino

  1. Connect your Arduino to the computer using a USB cable.
  2. Open Arduino IDE and paste the above code.
  3. Select the correct board and port:
  • Go to Tools → Board → Select your board (e.g., Arduino Uno).
  • Go to Tools → Port → Select the correct COM port.
  1. Click Upload (➡️) to flash the code to the Arduino.

Troubleshooting Tips

1. OLED Not Displaying Anything?

  • Double-check the I2C wiring (SDA → A4, SCL → A5).
  • Make sure the OLED module uses 0x3C as its I2C address.
  • If unsure, use an I2C scanner sketch to check the correct address.

2. Library Errors?

  • Ensure you've installed the Adafruit GFX and Adafruit SSD1306 libraries.
  • If errors persist, reinstall the libraries using the Arduino Library Manager.

3. Screen Flickering?

  • If the screen flickers, reduce the update frequency or check power stability.
