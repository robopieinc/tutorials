🎹 Arduino Buzzer and Button-Based Music Player (Sa Re Ga Ma Piano)

📘 Project Overview:

In this interactive electronics project, students will build a simple piano using an Arduino Uno, 8 push buttons, and a passive buzzer. Each button corresponds to a note in the Indian classical scale: Sa, Re, Ga, Ma, Pa, Dha, Ni, Sa (High Octave). When a button is pressed, the Arduino outputs a frequency to the buzzer to generate the corresponding musical tone.

This project is ideal for beginners to learn digital input/output, basic sound generation, and circuit building on a breadboard. It’s also a fun way to combine creativity and technology.

🔌 Circuit Theory and Explanation:

🟢 Buttons (Digital Inputs):

Each button is connected to a separate Arduino digital pin (Pins 6 through 13). When a button is pressed, it completes a circuit that sends a HIGH (5V) signal to the Arduino pin.

To ensure stability, each button also has a 10kΩ pull-down resistor to GND. This prevents the pin from "floating" (unpredictable behavior) when the button is not pressed.

🔊 Buzzer (Sound Output):

The passive buzzer is connected to Pin 2 of the Arduino. Unlike active buzzers, passive buzzers require specific frequency signals to generate different tones. The tone() function is used to send a square wave of a specific frequency to the buzzer, producing musical notes.

🔌 Power Supply:

All buttons are supplied with 5V from the Arduino board. One side of each button is connected to 5V, and the other side goes to an Arduino pin and a 10kΩ resistor connected to GND.

🖨️ Circuit Diagram Summary:

➤ Buzzer:

  • Positive terminal → Pin 2
  • Negative terminal → GND

➤ Push Buttons:

  • Buttons connected to Pins 6 through 13
  • One terminal → 5V
  • Other terminal → respective Arduino pin + 10kΩ resistor → GND

💻 Arduino Code:

int buz = 2; // Define the buzzer pin

int a = 6; // Sa

int b = 7; // Re

int c = 8; // Ga

int d = 9; // Ma

int e = 10; // Pa

int f = 11; // Dha

int g = 12; // Ni

int h = 13; // Sa (High)

void setup() {

  // Set the buzzer pin as an output

  pinMode(buz, OUTPUT);

 

  // Set the button pins as inputs

  pinMode(a, INPUT);

  pinMode(b, INPUT);

  pinMode(c, INPUT);

  pinMode(d, INPUT);

  pinMode(e, INPUT);

  pinMode(f, INPUT);

  pinMode(g, INPUT);

  pinMode(h, INPUT);

  // Initialize the buzzer to be off

  digitalWrite(buz, LOW);

}

void loop() {

  // Define the duration for each note

  int duration = 700;

  int dura = 500;

  // Read the state of each button

  int sa = digitalRead(a);

  int re = digitalRead(b);

  int ga = digitalRead(c);

  int ma = digitalRead(d);

  int pa = digitalRead(e);

  int dha = digitalRead(f);

  int ni = digitalRead(g);

  int sasa = digitalRead(h);

  // Play the corresponding tone based on the button press

  if (sa == 1) {

    tone(buz, 261.63, dura); // Play Sa

    delay(duration);

  } else if (re == 1) {

    tone(buz, 293.66, dura); // Play Re

    delay(duration);

  } else if (ga == 1) {

    tone(buz, 329.63, dura); // Play Ga

    delay(duration);

  } else if (ma == 1) {

    tone(buz, 349.23, dura); // Play Ma

    delay(duration);

  } else if (pa == 1) {

    tone(buz, 392.00, dura); // Play Pa

    delay(duration);

  } else if (dha == 1) {

    tone(buz, 440.00, dura); // Play Dha

    delay(duration);

  } else if (ni == 1) {

    tone(buz, 493.88, dura); // Play Ni

    delay(duration);

  } else if (sasa == 1) {

    tone(buz, 523.25, dura); // Play Sa (Higher Octave)

    delay(duration);

  }

}

🔍 Code Explanation:

  • Pin Declarations: Each pin is assigned to either a button or the buzzer.
  • setup() Function: Initializes the pins. Buttons are inputs, the buzzer is an output.
  • loop() Function: Continuously checks which button is pressed.
  • If a button is detected as HIGH, the corresponding note frequency is played using the tone() function.
  • A small delay() after each tone ensures clarity between notes.

🧠 Learning Outcomes:

  • Understanding digital inputs (buttons) and output (buzzer).
  • Working with tone generation using PWM on Arduino.
  • Applying basic circuit design with pull-down resistors.
  • Exploring the musical scale through electronics.

👣 Step-by-Step Instructions for Students:

  1. Assemble the Circuit:
  • Follow the pin mapping and wiring diagram to connect the buttons and buzzer to the Arduino.
  1. Upload the Code:
  • Open the Arduino IDE, paste the provided code, select the correct COM port, and upload the sketch.
  1. Test the Piano:
  • Press each button to hear the corresponding note on the buzzer.
  1. Experiment:
  • Try changing the frequencies to create custom melodies.
  • Add LEDs or an OLED screen to display the note being played.
