Tutorial 2: Reading Data from Firebase Using Raspberry Pi

Objective: Learn how to retrieve data from Firebase's Realtime Database using a Raspberry Pi and Python.

Prerequisites:

  • Raspberry Pi with internet connectivity
  • Python installed on the Raspberry Pi
  • A Firebase project with Realtime Database containing data

Steps:

  1. Set Up Firebase Project:
  • Ensure you have a Firebase project with the Realtime Database populated with data you wish to retrieve.
  1. Install Required Python Libraries:

If not already installed, install the pyrebase library:

pip install pyrebase4

  1. Configure Firebase in Python:

Create a Python script (e.g., firebase_read.py) with the following content:

import pyrebase

# Firebase configuration

config = {

    "apiKey": "YOUR_API_KEY",

    "authDomain": "YOUR_PROJECT_ID.firebaseapp.com",

    "databaseURL": "https://YOUR_PROJECT_ID.firebaseio.com",

    "storageBucket": "YOUR_PROJECT_ID.appspot.com"

}

# Initialize Firebase

firebase = pyrebase.initialize_app(config)

db = firebase.database()

# Retrieve data from Firebase

sensor_data = db.child("sensor_data").get()

if sensor_data.each():

    for data in sensor_data.each():

        print(data.key(), data.val())

else:

    print("No data available.")

  •  Explanation:
  • Replace placeholders like YOUR_API_KEY and YOUR_PROJECT_ID with actual values from your Firebase project.
  • The script retrieves data from the sensor_data node.
  • It iterates through each child in sensor_data and prints the key and value.
  1. Run the Script:

Execute the script:

python firebase_read.py

  • If data exists in the specified node, it will be printed to the console. Otherwise, a message indicating no data is available will be displayed.
