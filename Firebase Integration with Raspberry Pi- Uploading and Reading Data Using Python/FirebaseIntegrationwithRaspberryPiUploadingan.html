Tutorial 1: Uploading Data to Firebase Using Raspberry Pi

Objective: Learn how to send data from a Raspberry Pi to Firebase's Realtime Database using Python.

Prerequisites:

  • Raspberry Pi with internet connectivity
  • Python is installed on the Raspberry Pi
  • A Firebase project with Realtime Database enabled

Steps:

  1. Set Up Firebase Project:
  • Navigate to the Firebase Console and create a new project.
  • Enable the Realtime Database for your project.

In the Database section, set the rules to allow read and write operations for testing purposes:

{

  "rules": {

    ".read": true,

    ".write": true

  }

}

  •  Note: For production, ensure you implement proper authentication and security rules.
  1. Install Required Python Libraries:

Install the pyrebase library, a Python wrapper for Firebase.

pip install pyrebase4

  1. Configure Firebase in Python:
  • Obtain your Firebase project's configuration details from the Firebase Console under Project Settings.

Create a Python script (e.g., firebase_upload.py) and include the following code:

import pyrebase

# Firebase configuration

config = {

    "apiKey": "YOUR_API_KEY",

    "authDomain": "YOUR_PROJECT_ID.firebaseapp.com",

    "databaseURL": "https://YOUR_PROJECT_ID.firebaseio.com",

    "storageBucket": "YOUR_PROJECT_ID.appspot.com"

}

# Initialize Firebase

firebase = pyrebase.initialize_app(config)

db = firebase.database()

# Data to be uploaded

data = {

    "temperature": 25.5,

    "humidity": 60

}

# Upload data to Firebase

db.child("sensor_data").push(data)

print("Data uploaded successfully.")

  •  Explanation:
  • Replace placeholders like YOUR_API_KEY and YOUR_PROJECT_ID with actual values from your Firebase project.
  • The data dictionary contains the sensor readings you wish to upload.
  • The push method adds the data to the sensor_data node in your Realtime Database.
  1. Run the Script:

Execute the script:

python firebase_upload.py

  • Upon successful execution, the data will be uploaded to Firebase, and you'll see the confirmation message.
