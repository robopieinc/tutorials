🔥 Firebase Data Upload & Download with ESP8266 and ESP32 – Full Tutorial

This tutorial guides you step-by-step through uploading (writing) and downloading (reading) data to and from Firebase Realtime Database using an ESP8266 or ESP32 board.

✅ Prerequisites

Before you begin, make sure you have:

  • An ESP8266 (e.g., NodeMCU) or ESP32 board
  • A Firebase account (free tier is enough)
  • A Firebase Realtime Database project set up
  • Arduino IDE installed with ESP8266/ESP32 board support
  • "Firebase_ESP_Client" library installed

🔧 Arduino IDE Setup

1. Install Board Support

  • Open Arduino IDE
  • Go to File > Preferences
  • In Additional Board Manager URLs, add:
  • For ESP32: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  • For ESP8266: http://arduino.esp8266.com/stable/package_esp8266com_index.json
  • Then go to Tools > Board > Boards Manager and install:
  • ESP32 by Espressif Systems
  • ESP8266 by ESP8266 Community

2. Install Required Libraries

  • Go to Sketch > Include Library > Manage Libraries
  • Search and install:
  • Firebase_ESP_Client by Mobizt
  • ArduinoJson (optional, but useful for JSON handling)

☁️ Firebase Setup

1. Create a Firebase Project

  • Go to Firebase Console
  • Click Add Project, name it, and follow the steps
  • In the Build section, go to Realtime Database
  • Click Create Database and choose a location

2. Set Database Rules for Testing

For testing purposes (not recommended for production), set the rules to public:

{

  "rules": {

    ".read": true,

    ".write": true

  }

}

⚠️ Don't forget to change these rules later for security!

3. Get Credentials

  • In Firebase console, go to Project Settings > General
  • Copy the Web API Key
  • In Realtime Database, copy the Database URL

📤 1. Upload (Write) Data to Firebase

✅ Code (ESP8266 / ESP32 – Writing)

#include <WiFi.h> // Use <ESP8266WiFi.h> for ESP8266

#include <Firebase_ESP_Client.h>

// WiFi Credentials

#define WIFI_SSID "your_SSID"

#define WIFI_PASSWORD "your_PASSWORD"

// Firebase Credentials

#define API_KEY "your_firebase_api_key"

#define DATABASE_URL "your_database_url"

FirebaseData fbdo;

FirebaseAuth auth;

FirebaseConfig config;

void setup() {

    Serial.begin(115200);

    WiFi.begin(WIFI_SSID, WIFI_PASSWORD);

    while (WiFi.status() != WL_CONNECTED) {

        delay(500);

        Serial.print(".");

    }

    Serial.println("\nConnected to WiFi");

   

    config.api_key = API_KEY;

    config.database_url = DATABASE_URL;

    Firebase.begin(&config, &auth);

    Firebase.reconnectWiFi(true);

}

void loop() {

    // Writing data

    if (Firebase.RTDB.setFloat(&fbdo, "/sensor/light", 75.5)) {

        Serial.println("Data written successfully!");

    } else {

        Serial.println("Failed: " + fbdo.errorReason());

    }

    delay(5000);

}

💡 How it works

  • Connects to WiFi
  • Connects to Firebase with your credentials
  • Writes a float value (75.5) to path /sensor/light every 5 seconds

📥 2. Download (Read) Data from Firebase

✅ Code (ESP8266 / ESP32 – Reading)

#include <WiFi.h> // Use <ESP8266WiFi.h> for ESP8266

#include <Firebase_ESP_Client.h>

#define WIFI_SSID "your_SSID"

#define WIFI_PASSWORD "your_PASSWORD"

#define API_KEY "your_firebase_api_key"

#define DATABASE_URL "your_database_url"

FirebaseData fbdo;

FirebaseAuth auth;

FirebaseConfig config;

void setup() {

    Serial.begin(115200);

    WiFi.begin(WIFI_SSID, WIFI_PASSWORD);

    while (WiFi.status() != WL_CONNECTED) {

        delay(500);

        Serial.print(".");

    }

    Serial.println("\nConnected to WiFi");

   

    config.api_key = API_KEY;

    config.database_url = DATABASE_URL;

    Firebase.begin(&config, &auth);

    Firebase.reconnectWiFi(true);

}

void loop() {

    // Reading data

    float lightValue;

    if (Firebase.RTDB.getFloat(&fbdo, "/sensor/light")) {

        lightValue = fbdo.floatData();

        Serial.print("Light Intensity: ");

        Serial.println(lightValue);

    } else {

        Serial.println("Failed: " + fbdo.errorReason());

    }

    delay(5000);

}

💡 How it works

  • Connects to WiFi and Firebase
  • Reads the value at /sensor/light
  • Displays it in Serial Monitor every 5 seconds

✅ Final Tips

  • Use Serial Monitor (baud: 115200) to debug and see outputs.
  • Make sure your WiFi SSID/password, API key, and database URL are correct.
  • If authentication is enabled, you’ll need to handle the auth.token with user sign-in or anonymous access.
  • Use different paths in Firebase for different sensors/data types.

📌 Conclusion

  • Use the first sketch to upload data to Firebase.
  • Use the second to download and read data from Firebase.
  • Works with both ESP8266 and ESP32.
  • Great for IoT dashboards, sensor monitoring, home automation, etc.
