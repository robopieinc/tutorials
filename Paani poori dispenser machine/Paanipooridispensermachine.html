#Paani poori dispenser machine code

Pani Poori Machine Tutorial Using Arduino

Objective:

To create a simple automated pani poori machine that dispenses pani (water) when an object (like a bowl or a hand) is detected by an infrared (IR) sensor.

Components Required:

Arduino Uno - The microcontroller that controls the entire process.

L298N Motor Driver Module - Used to control the DC motor that will pump the pani.

DC Motor - Connected to a pump to dispense pani.

IR Sensor Module - Detects objects like a hand or bowl to trigger the motor.

Breadboard and Jumper Wires - For making connections.

Power Supply - To power the Arduino and motor driver.

Other components, such as LED indicators and resistors (if needed).

Circuit Diagram:

Refer to the provided circuit diagram for wiring connections.

IR Sensor Connections:

VCC to Arduino 5V.

GND to Arduino GND.

OUT to Arduino Digital Pin 2.

L298N Motor Driver Connections:

ENA to Arduino Digital Pin 6.

IN1 to Arduino Digital Pin 6.

IN2 to Arduino Digital Pin 5.

Motor terminals connected to the DC motor.

Arduino to Power Supply:

Connect the 5V and GND pins of Arduino to the breadboard's power rails.

Code Explanation:

int ir = 2;

int a = 6; // motor ka a1

int b = 5; // motor ka a2;

void setup() {

 pinMode(ir,INPUT);

 pinMode(a,OUTPUT);

 pinMode(b,OUTPUT);

 Serial.begin(9600);

}

void loop() {

 int x = digitalRead(ir);

 Serial.println(x);

 if (x==0){

  digitalWrite(a,HIGH);

  digitalWrite(b,LOW);

  delay(1000);

 }

 else if (x==1){

  digitalWrite(a,LOW);

  digitalWrite(b,LOW);

 }

 delay(10);

}

Working Principle:

  1. The IR sensor continuously checks for an object. If it detects something (like a bowl or a hand), it sends a LOW signal (0) to the Arduino.
  2. When the Arduino receives this LOW signal, it activates the motor via the L298N motor driver to pump pani for a specific duration (1 second in this code).
  3. If no object is detected, the motor remains OFF.

Adjustments:

  • You can change the delay(1000) value to adjust how long the motor runs.
  • Ensure that the power supply to the motor driver and Arduino is adequate to run the motor without interruptions.

Safety Note:

  • Always double-check your wiring connections before powering up the circuit to avoid short circuits or damaging components.

Conclusion:

This simple setup allows you to create an automated pani-poori machine using basic electronics and programming with Arduino. You can further enhance it by adding more sensors or control elements.
