🚀 Arduino Joystick Controlled LED Project – Full Tutorial

🧠 Objective:

Control 5 different LEDs using a joystick module. Based on the direction the joystick is pushed, a specific LED will light up. If the joystick is centered, all LEDs turn OFF. This helps students understand:

  • Analog input reading (analogRead)
  • Digital input reading (digitalRead)
  • Conditional logic (if / else if)
  • Controlling outputs (digitalWrite)
  • Debouncing behavior with delay()

🧾 Arduino Code (Clean & Structured):

int vx = A0;   // X-axis

int vy = A1;   // Y-axis

int SW = 2;    // Joystick button (switch)

int a = 3;     // LED A – Right

int b = 5;     // LED B – Left

int c = 6;     // LED C – Press

int d = 9;     // LED D – Up

int e = 11;    // LED E – Down

void setup() {

  pinMode(vx, INPUT);

  pinMode(vy, INPUT);

  pinMode(SW, INPUT); // Button is HIGH when pressed

  pinMode(a, OUTPUT);

  pinMode(b, OUTPUT);

  pinMode(c, OUTPUT);

  pinMode(d, OUTPUT);

  pinMode(e, OUTPUT);

  Serial.begin(9600);

}

void loop() {

  int x = analogRead(vx);

  int y = analogRead(vy);

  int z = digitalRead(SW); // LOW when pressed

  Serial.print("vx=");

  Serial.print(x);

  Serial.print("  vy=");

  Serial.print(y);

  Serial.print("  SW=");

  Serial.println(z);

  if (x > 600) {

    // Joystick moved RIGHT

    digitalWrite(a, HIGH);

    digitalWrite(b, LOW);

    digitalWrite(c, LOW);

    digitalWrite(d, LOW);

    digitalWrite(e, LOW);

    delay(1000);

  }

  else if (x < 400) {

    // Joystick moved LEFT

    digitalWrite(a, LOW);

    digitalWrite(b, HIGH);

    digitalWrite(c, LOW);

    digitalWrite(d, LOW);

    digitalWrite(e, LOW);

    delay(1000);

  }

  else if (z == 1) {

    // Joystick is pressed

    digitalWrite(a, LOW);

    digitalWrite(b, LOW);

    digitalWrite(c, HIGH);

    digitalWrite(d, LOW);

    digitalWrite(e, LOW);

    delay(1000);

  }

  else if (y > 600) {

    // Joystick moved UP

    digitalWrite(a, LOW);

    digitalWrite(b, LOW);

    digitalWrite(c, LOW);

    digitalWrite(d, HIGH);

    digitalWrite(e, LOW);

    delay(1000);

  }

  else if (y < 400) {

    // Joystick moved DOWN

    digitalWrite(a, LOW);

    digitalWrite(b, LOW);

    digitalWrite(c, LOW);

    digitalWrite(d, LOW);

    digitalWrite(e, HIGH);

    delay(1000);

  }

  else {

    // Joystick in neutral position

    digitalWrite(a, LOW);

    digitalWrite(b, LOW);

    digitalWrite(c, LOW);

    digitalWrite(d, LOW);

    digitalWrite(e, LOW);

  }

  delay(10); // Small delay to stabilize readings

}

📊 Understanding the Analog Range:

  • Joystick Center (Neutral):
  • X: around 500
  • Y: around 500
  • Thresholds:
  • RIGHT: x > 600
  • LEFT: x < 400
  • UP: y > 600
  • DOWN: y < 400
  • PRESS: z == LOW (because of pull-up)

📌 Tips for Students:

  • Always turn off unused LEDs to avoid multiple ON states.
  • Use Serial Monitor (9600 baud) to debug values in real-time.
  • Use INPUT_PULLUP for joystick switch to avoid using an external resistor.
  • You can change pin numbers or thresholds based on your joystick behavior.
