💡 Understanding the Components

🔴 LED (Light Emitting Diode)

  • A small light that glows when powered correctly.
  • Has two legs:
  • Anode (Long leg) → Connects to positive (Arduino pin).
  • Cathode (Short leg) → Connects to ground (GND).

🔌 Breadboard

  • Allows us to connect components without soldering.
  • Rows (A–E, F–J) are connected horizontally.
  • Power rails (side columns) are connected vertically.
  • Used to organize and test electronic circuits easily.

🔧 Wiring Instructions

Here’s how to connect your circuit:

  1. Place 4 LEDs on the breadboard in separate rows.
  2. Connect a 220Ω resistor to the anode (long leg) of each LED.
  3. Connect each resistor to Arduino pins 2, 3, 4, and 5 respectively.
  4. Connect all cathodes (short leg of LED) to GND rail.
  5. Finally, connect Arduino GND to the breadboard GND rail.

💡 This setup lets Arduino control the LEDs safely through digital pins.

🧪 Project 1: Blinking 1 LED

// Define the pin where the LED is connected

int led1 = 2;

void setup() {

  // Set pin 2 as OUTPUT so it can send voltage to the LED

  pinMode(led1, OUTPUT);

}

void loop() {

  // Turn the LED ON by sending HIGH voltage

  digitalWrite(led1, HIGH);

  delay(500); // Wait for 500 milliseconds

  // Turn the LED OFF by sending LOW voltage

  digitalWrite(led1, LOW);

  delay(500); // Wait again

}

🧪 Project 2: Blinking 2 LEDs Alternately

int led1 = 2; // LED 1 on pin 2

int led2 = 3; // LED 2 on pin 3

void setup() {

  pinMode(led1, OUTPUT); // Set pin 2 as OUTPUT

  pinMode(led2, OUTPUT); // Set pin 3 as OUTPUT

}

void loop() {

  digitalWrite(led1, HIGH); // LED 1 ON

  digitalWrite(led2, LOW);  // LED 2 OFF

  delay(500);               // Wait for 500ms

  digitalWrite(led1, LOW);  // LED 1 OFF

  digitalWrite(led2, HIGH); // LED 2 ON

  delay(500);               // Wait for 500ms

}

🧪 Project 3: Blinking 3 LEDs in Sequence

int led1 = 2;

int led2 = 3;

int led3 = 4;

void setup() {

  // Set all pins as OUTPUT

  pinMode(led1, OUTPUT);

  pinMode(led2, OUTPUT);

  pinMode(led3, OUTPUT);

}

void loop() {

  // Turn ON LED 1

  digitalWrite(led1, HIGH);

  delay(300);

  digitalWrite(led1, LOW);

  // Turn ON LED 2

  digitalWrite(led2, HIGH);

  delay(300);

  digitalWrite(led2, LOW);

  // Turn ON LED 3

  digitalWrite(led3, HIGH);

  delay(300);

  digitalWrite(led3, LOW);

}

🧪 Project 4: Blink 4 LEDs Together

int led1 = 2;

int led2 = 3;

int led3 = 4;

int led4 = 5;

void setup() {

  // Initialize all LED pins as OUTPUT

  pinMode(led1, OUTPUT);

  pinMode(led2, OUTPUT);

  pinMode(led3, OUTPUT);

  pinMode(led4, OUTPUT);

}

void loop() {

  // Turn ON all LEDs

  digitalWrite(led1, HIGH);

  digitalWrite(led2, HIGH);

  digitalWrite(led3, HIGH);

  digitalWrite(led4, HIGH);

  delay(500);

  // Turn OFF all LEDs

  digitalWrite(led1, LOW);

  digitalWrite(led2, LOW);

  digitalWrite(led3, LOW);

  digitalWrite(led4, LOW);

  delay(500);

}
