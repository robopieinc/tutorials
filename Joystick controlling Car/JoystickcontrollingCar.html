🚗✨ Joystick Controlled Arduino Car + Laser/Buzzer/LED – Full Tutorial

🎯 Objective:

Control a 2-motor car using a joystick module, and activate an LED/buzzer/laser when the joystick is pressed. Learnings include:

  • Reading joystick analog/digital inputs
  • Controlling DC motors using digital outputs
  • Understanding basic motor directions: forward, reverse, left, right
  • Using a separate pin for a shooter (laser/LED/buzzer)

🔧 Arduino Code: Improved & Structured

int urx = A0; // X-axis

int ury = A1; // Y-axis

int sw = A2;  // Switch press

int a = 2;    // Motor A1

int b = 3;    // Motor A2

int c = 4;    // Motor B1

int d = 5;    // Motor B2

int p = 12;   // Shooter (LED / Laser / Buzzer)

void setup() {

  pinMode(urx, INPUT);

  pinMode(ury, INPUT);

  pinMode(sw, INPUT_PULLUP); // Button is LOW when pressed

  pinMode(a, OUTPUT);

  pinMode(b, OUTPUT);

  pinMode(c, OUTPUT);

  pinMode(d, OUTPUT);

  pinMode(p, OUTPUT);

  Serial.begin(9600);

}

void loop() {

  int x = analogRead(urx);

  int y = analogRead(ury);

  int z = digitalRead(sw); // LOW when pressed

  Serial.print("X = ");

  Serial.print(x);

  Serial.print(" | Y = ");

  Serial.print(y);

  Serial.print(" | SW = ");

  Serial.println(z);

  if (x < 400) {

    // Move RIGHT

    Serial.println("Right Movement");

    digitalWrite(a, HIGH);

    digitalWrite(b, LOW);

    digitalWrite(c, HIGH);

    digitalWrite(d, LOW);

  }

  else if (x > 600) {

    // Move LEFT

    Serial.println("Left Movement");

    digitalWrite(a, LOW);

    digitalWrite(b, HIGH);

    digitalWrite(c, LOW);

    digitalWrite(d, HIGH);

  }

  else if (y < 400) {

    // Move FORWARD

    Serial.println("Forward Movement");

    digitalWrite(a, LOW);

    digitalWrite(b, HIGH);

    digitalWrite(c, HIGH);

    digitalWrite(d, LOW);

  }

  else if (y > 600) {

    // Move BACKWARD

    Serial.println("Backward Movement");

    digitalWrite(a, HIGH);

    digitalWrite(b, LOW);

    digitalWrite(c, LOW);

    digitalWrite(d, HIGH);

  }

  else if (z == LOW) {

    // Fire shooter

    Serial.println("Shooter Activated!");

    digitalWrite(p, HIGH);

  }

  else {

    // Stop all motors and shooter

    digitalWrite(a, LOW);

    digitalWrite(b, LOW);

    digitalWrite(c, LOW);

    digitalWrite(d, LOW);

    digitalWrite(p, LOW);

  }

  delay(300);

}

🧠 Working Explanation:

  • Joystick Centered: Stops the car.
  • X-axis:
  • Left (<400): Right turn
  • Right (>600): Left turn
  • Y-axis:
  • Up (<400): Forward
  • Down (>600): Reverse
  • SW Pressed: Turns ON pin 12 (used for buzzer/LED/laser).

📌 Suggestions for Enhancement:

  • Replace direct motor control with L298N motor driver for better power handling.
  • Add PWM pins (e.g., 5, 6, 9, 10) for speed control using analogWrite.
  • Use Li-ion battery pack to run the motors independently of Arduino power.
