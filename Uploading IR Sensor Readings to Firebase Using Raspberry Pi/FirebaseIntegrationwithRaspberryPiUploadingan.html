Tutorial 3: Uploading IR Sensor Readings to Firebase Using Raspberry Pi

Objective:
 Learn how to read data from an IR sensor and upload it to Firebase in real time using a Raspberry Pi and Python.

Prerequisites:

  • Raspberry Pi with internet access
  • IR Sensor (any reflective IR sensor like TCRT5000)
  • Firebase project with Realtime Database enabled
  • Python installed on Raspberry Pi
  • Resistor (if needed for sensor connection)

Connections:

  • IR Sensor Pins:
  • VCC → 5V or 3.3V (depending on sensor)
  • GND → Ground
  • OUT → Raspberry Pi GPIO pin (e.g., GPIO 17)

Step 1: Install Required Libraries

Run the following command to install the necessary Python library:

pip install pyrebase4 RPi.GPIO

Step 2: Configure Firebase

  • Go to the Firebase Console and create a new project.
  • Enable the Realtime Database.

Set read/write rules for testing:

{

  "rules": {

    ".read": true,

    ".write": true

  }

}

Step 3: Python Code for Uploading IR Sensor Data to Firebase

Create a Python file ir_to_firebase.py and add the following code:

import pyrebase

import RPi.GPIO as GPIO

import time

# Firebase configuration

config = {

    "apiKey": "YOUR_API_KEY",

    "authDomain": "YOUR_PROJECT_ID.firebaseapp.com",

    "databaseURL": "https://YOUR_PROJECT_ID.firebaseio.com",

    "storageBucket": "YOUR_PROJECT_ID.appspot.com"

}

# Initialize Firebase

firebase = pyrebase.initialize_app(config)

db = firebase.database()

# Setup Raspberry Pi GPIO

IR_PIN = 17  # Use GPIO pin 17

GPIO.setmode(GPIO.BCM)

GPIO.setup(IR_PIN, GPIO.IN)

try:

    while True:

        ir_value = GPIO.input(IR_PIN)  # Read sensor value

        data = {"IR_Sensor": ir_value}

        db.child("sensor_data").set(data)  # Upload to Firebase

        print(f"IR Sensor Value: {ir_value} uploaded to Firebase")

        time.sleep(1)  # Send data every second

except KeyboardInterrupt:

    GPIO.cleanup()

    print("Program stopped.")

Step 4: Run the Script

Execute the script:

python ir_to_firebase.py

Expected Output:

  • If the IR sensor detects an object, it will upload 1 to Firebase.
  • If no object is detected, it will upload 0.
  • Data will be visible under the sensor_data node in Firebase.
